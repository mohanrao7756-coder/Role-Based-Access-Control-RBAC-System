# Role-Based-Access-Control-RBAC-System
1.Project Introduction:

This project is a Role-Based Access Control (RBAC) System developed using Spring Boot.

The main goal of this project is to implement secure authentication and authorization using JWT tokens.

The system allows:
- User Registration
- User Login
- JWT Token Authentication
- Role Management
- Permission Management
- Secured APIs using Spring Security

2.Technologies Used:

Technologies used:
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- JWT
- H2 Database
- Maven
- Postman


3.Project Architecture:

I followed layered architecture for clean code and maintainability.

Layers:
1. Controller Layer
2. Service Layer
3. Repository Layer
4. Entity Layer
5. Security Layer
6. Exception Layer

4.I used Postman to test REST APIs:

Testing steps:
- Register user
- Login
- Copy JWT token
- Pass token in Authorization header
- Access secured APIs
